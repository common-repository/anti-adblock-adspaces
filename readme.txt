== Installation ==

This section describes how to install the plugin and get it working.

1. Visit 'Plugins > Add New'
2. Upload WP Multi-User Anti-AdBlocker Widget.zip
3. Activate Anti Adblock Adspaces from your Plugins page.
4. The widget will now be available in the 'Appearance -> Widgets' page, you can add the new Anti Adblock Adspaces widget to a sidebar of your Theme.
